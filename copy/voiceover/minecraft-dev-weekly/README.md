# Minecraft 开发者周报 TTS 区

## 目录规则

- `YYYY-WW-segment-01.tts.md`：分段 TTS 文本。
- `YYYY-WW-manifest.md`：本期语音素材清单。

## MiniMax 默认参数

- profile：`audio`
- key source：`MINIMAX_AUDIO_KEY`
- voice：`Chinese (Mandarin)_Laid_BackGirl`
- speed：`1.12`
- pitch：`0`
- mix：空
- model：`speech-2.8-hd`

## 发音规则

- `KubeJS` 不写成 `kubejs`，除非需要强调真实目录名且已确认发音可接受。
- `NeoForge`、`Forge`、`Fabric`、`Quilt` 保持英文写法。
- `Modrinth`、`CurseForge` 保持英文写法。
- MC 版本读法可以保留字幕原文，TTS 里按自然中文读。
