# v07 情绪拼接样例

状态：测评语音实验。样例文案不代表真实整合包。

## 结构

| Part | 情绪 | 文本 |
| --- | --- | --- |
| 01 | `angry` | 我一开始是真的有点火大。 |
| 02 | `angry` | 不是因为它做得差，而是它明明有很好的底子。 |
| 03 | `sad` | 可它总在最关键的时候，替玩家把决定做完。 |
| 04 | `happy` | 但玩到最后，我反而慢慢释怀了。 |
| 05 | 无 | 它不是没把路做好，只是从一开始，就没打算把路交给我自己走。 |

## 合成目标

先用愤怒建立冲突，再转到无奈，最后落到释怀和判断。
